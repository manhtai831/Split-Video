Chọn một hoặc nhiều video. Mỗi video sẽ tạo một job riêng. Hỗ trợ MP4, MOV, MKV, …

File đã chọn

MP3 và M4A phù hợp hầu hết trình phát. WAV/FLAC giữ chất lượng gốc, file lớn hơn.

Original nhanh nhất khi định dạng trùng codec nguồn. Không áp dụng cho WAV/FLAC.

Tùy chọn nâng cao

100% = giữ nguyên. 0% = im lặng. 200% = gấp đôi âm lượng.

Tăng tốc độ làm file ngắn hơn; giảm tốc độ làm file dài hơn. Giữ nguyên pitch.

Title tự động = tên file gốc (bỏ phần mở rộng) cho từng job.
